Задание 2: проектирование API
Что нужно сделать:
1.	Написать пример REST API запроса, который будет вызываться при переходе пользователя на данный экран. 
При открытии экрана мобильное приложение отправляет GET-запрос на сервер для получения списка партнёрских магазинов.
Запрос передаёт:
- город пользователя
- координаты (если доступны)
- язык интерфейса
- токен авторизации
Метод: GET 

2.	Привести пример ответа этого REST API в соответствии с макетом. Формат - JSON. Учесть, что при клике на плашку магазина должен осуществляться переход по ссылке на внешний ресурс. 
Сервер возвращает JSON-объект, содержащий:
- идентификатор запроса
- дату генерации ответа
- список магазинов
Каждый магазин содержит:
- store_id — уникальный идентификатор
- title — название магазина
- logo_url — ссылка на логотип
- delivery — информация о доставке: тип доставки (слот или быстрая), текст отображения, время доставки
- external_url — ссылка для перехода при клике
